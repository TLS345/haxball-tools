# haxball-tools-32

Track goals, assists, and own goals in real-time and announce the MVP at the end of the match in Haxball Headless rooms.

## Features
- Tracks goals, real assists, and own goals
- Calculates MVP automatically when the game ends
- Lightweight and easy to use
- Works in Haxball Headless

## How to use
1. Copy `mvp-tracker.js` to your project
2. Open a Haxball Headless room
3. Run the script
