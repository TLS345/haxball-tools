# Day 15/60 - Password Control
By TLS / Teleese

This script lets Haxball room hosts change or remove the room password while the room is running.

### Commands
- `!admin <password>` → Grants admin rights.
- `!pass 1234` / `!newpass 1234` → Sets the room password.
- `!pass off` / `!pass remove` → Removes the current password.
- `!pass` → Shows current password status.

### Notes
Only admins can change or remove passwords.  
Non-admin players can only check if a password exists.
